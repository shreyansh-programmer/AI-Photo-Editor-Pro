## 🎓 First Time Users

1. **Open the Application**
   - Double-click `AIPhotoEditorPro.exe`
   - Wait for it to load (first launch takes ~5 seconds)

2. **Import a Photo**
   - Click "File" → "Open"
   - Select your image

3. **Use AI Assistant**
   - Click the "Copilot" panel on the right
   - Type natural language commands like:
     - "Enhance the sky"
     - "Increase contrast"
     - "Apply warm tones"

4. **Export Your Work**
   - Click "File" → "Export"
   - Choose format and quality
   - Save to your desired location

## ❓ Troubleshooting

### Application Won't Start
- Ensure Windows Defender allows the app
- Try running as Administrator
- Check system requirements

### Slow Performance
- Close other applications
- Reduce image resolution
- Enable GPU acceleration in settings

### Image Not Loading
- Check file format (JPG, PNG, TIFF supported)
- Verify file is not corrupted
- Try a different image

### Installation

**Windows**: Download and run the .exe - no installation needed!

**Linux**:
```bash
chmod +x AIPhotoEditorPro
./AIPhotoEditorPro
```

**From Source**:
```bash
git clone https://github.com/shreyansh-programmer/AI-Photo-Editor-Pro.git
cd AI-Photo-Editor-Pro
pip install -r requirements.txt
python main.py
```
